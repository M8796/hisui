树

说明:

树形菜单（Tree）可从标记创建，也可定义在 <ul> 元素中。无序列表的 <ul> 元素提供一个基础的树（Tree）结构。每一个 <li> 元素将产生一个树节点，子 <ul> 元素将产生一个父树节点。

如：

  • Folder
    • Sub Folder 1
      • File 11
      • File 12
      • File 13
    • File 2
    • File 3
  • Folder
    • Sub Folder 1
      • File 11
      • File 12
      • File 13
    • File 2
    • File 3
  • Folder
    • Sub Folder 1
      • File 11
      • File 12
      • File 13
    • File 2
    • File 3
