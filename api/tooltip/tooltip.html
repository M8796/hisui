提示

说明:

当用户点击或者鼠标悬停在此区域，将会显示提示。

如：

鼠标移动到这(点击试试)
鼠标移动到这 
<a id="tt1" href="#" title="这是提示信息" class="hisui-tooltip" data-options="position:'right'">鼠标移动到这(点击试试)</a>
<div style="margin:20px 0"></div>
<a id="tt" href="javascript:void(0)">鼠标移动到这</a>
<script type="text/javascript">
$(function(){
	var tt = $HUI.tooltip('#tt',{
		content:'<span style="color:green">这是通过js提示的</span>',
		position: 'bottom' //top , right, bottom, left
	});
	$("#tt1").click(function(){
		tt.show();  //显示下面的提示
	});
});
$(init);
</script>
