链接按钮

说明:

链接按钮（Link Button）是使用 <a>元素来创建的，所以实际上一个链接按钮（Link Button）就是一个显示为按钮样式的 <a>元素。

如：

查询 禁用 禁用 查询按钮 查询按钮 清屏哈哈哈 哈哈 确定

病人信息条处 hover颜色需要深一些 为其增加类 hover-dark

呵呵 哈哈 
<a href="#" class="hisui-linkbutton" data-options="iconCls:'icon-search',plain:true">查询</a>
<a href="#" class="hisui-linkbutton" data-options="iconCls:'icon-search',plain:true" disabled>禁用</a>
<a href="#" class="hisui-linkbutton" data-options="iconCls:'icon-search'" disabled>禁用</a>
<a href="#" class="hisui-linkbutton" data-options="iconCls:'icon-search'">查询按钮</a>
<a href="#" class="hisui-linkbutton" data-options="iconImg:'update.png'">清屏</a>
<a href="#" class="hisui-linkbutton" id="btn1">确定</a>
<p>病人信息条处</p>
<a href="#" class="hisui-linkbutton hover-dark" data-options="iconCls:'icon-search'">呵呵</a>
<a href="#" class="hisui-linkbutton hover-dark" >哈哈</a>

 }

大图按钮

说明:

在class中加入big， 同时data-options中加入plain:true可实现大图按钮功能。 大图标定义请查看图标(icon)界面

如：

保存 打印 删除 手工解锁
手工解锁 
<a class="hisui-linkbutton big" data-options="iconCls:'icon-big-unlock',plain:true">手工解锁</a>
<a class="hisui-linkbutton big" data-options="iconImg:'unlock.png',plain:true">手工解锁</a>
