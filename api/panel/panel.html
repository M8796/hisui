面板

说明:

面板是其他组件或元素的容器

如：

卡片类型面板 
<div class="hisui-panel" title="登记" style="height:150px;width:400px;padding:10px 30px 20px 30px" data-options="headerCls:'panel-header-card'"
>卡片类型面板</div>
灰色卡片面板只需修改"headerCls:'panel-header-card-gray'" 
<div class="hisui-panel" title="病人信息" style="width:400px;padding:10px 30px 20px 30px" data-options="iconCls:'icon-save',
closable:true,collapsible:true,minimizable:true,maximizable:true"></div>
灰色面板只需修改headerCls:'panel-header-gray'
没有图标怎么样？headerCls:'panel-header-gray'
蓝色面板只需修改headerCls:'panel-header-blue'
大一点面板headerCls:'panel-header-big'
大一点面板 headerCls:'panel-header-big panel-header-gray'
没标题 没有title
还可通过js生成 
<div id="accPanel" title="病人信息" style="width:400px;padding:10px 30px 20px 30px;">还可通过js生成</div>
<script type="text/javascript">
var init = function () {
	var valbox = $HUI.panel("#accPanel",{
		iconCls:'icon-save',
		closable:true,
		onBeforeClose:function(){alert("你点击了关闭!");return false},
		collapsible:true,
		onBeforeCollapse:function(){alert("你点击了收起!");return false},
		minimizable:true,
		onMinimize:function(){alert("你点击了最小化!");return false},
		maximizable:true,
		onMaximize:function(){alert("你点击了最大化!");return false},
		headerCls:'panel-header-acc'
	});
}
$(init);
</script>
